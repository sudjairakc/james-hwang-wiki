## Dance Skill: Precision Street Hybrid (Animation + Popping based)

| Category | Detail |
|---------|--------|
| Primary Style | Animation / Robot / Popping / Waving (สาย illusion—คุมร่างกายละเอียด แยกส่วนชัด) |
| Secondary Style | Lite Breaking + Tricking (ใช้เพิ่ม impact เป็นช่วง ๆ) |
| Freestyle Mode | Conceptual / Musical precision (ฟรีสไตล์แบบคิดเป็นภาพ เล่นกับจังหวะหน่วง/เร่ง) |
| Signature Control | Micro-isolation control + glitch/stop-time movement + delayed hit |
| Footwork Language | Clean footwork & glide techniques (การใช้เท้าคม เช่น glide / slide / pivot) |
| Groove Integration | Groove-based stepping with precision timing (สเต็ปเท้ามี groove แต่ยังคุมจังหวะเป๊ะ) |
| Upper/Lower Contrast | Upper-body isolation vs smooth lower-body stepping |
| Core Strength | Body control สูงมาก + illusion movement + footwork ที่ลื่นแต่คม |
| Limitation | ไม่ใช่สาย power ต่อเนื่องหรือ battle หนัก ๆ |
