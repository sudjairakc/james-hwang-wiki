Vocal Type: TenorVocal Range: Approximately C3 to B4 or C5, with falsetto extending to E5
Key Features:

* Soft and polished vocal tone (โทนนุ่มคลีน แต่เพิ่ม texture เล็กน้อย ไม่เรียบเกิน)
* Exceptional vocal control (คุมลมดี + เริ่ม “เล่นจังหวะเสียง” มากขึ้น ไม่ร้องตรงตลอด)
* Powerful mid-range: G3–E4 (ยังเป็นแกนหลัก แต่ใส่ groove และ accent มากขึ้น)
* Versatile tonal quality (จากเดิมที่นุ่ม → ตอนนี้มีทั้ง soft + rhythmic + percussive vocal)

Vocal Style Adjustment:
* Rhythmic phrasing (การ “วางคำตามจังหวะ” แบบ groove-based ไม่ใช่ร้องตรงเมโลดี้เป๊ะตลอด)
* Use of vocal accents & attacks (ใส่ลูกกระแทกเสียงสั้น ๆ เช่น “uh”, “ha”, “tch”)
* Controlled breathiness (ปล่อยลม intentionally ให้เสียงมี texture เซ็กซี่/ลอย)
* Dynamic push-pull timing (เล่นจังหวะหน่วง/เร่ง คล้ายการเต้น)
* Emotional sharpness (บางคำจะ “คม/พุ่ง” แทนที่จะนุ่มทั้งหมด)

Vocal Style Reference:

* Dean (R&B phrasing + groove)
* Crush (tone warmth + musicality)

Production Preferences:
* Focus on pop & R&B (แกนยังเป็นเพลงตลาด ฟังง่าย แต่มีคุณภาพโปรดักชันสูง)
* Smooth rhythms & rich melodic layering (จังหวะลื่น เมโลดี้มีเลเยอร์ชัด)
* Groove-centric arrangement (ให้ความสำคัญกับ rhythm section—กลอง/เบส เป็นตัวขับเพลง)
* MJ-inspired rhythm design: ใช้ drum pattern ที่ “เด้ง/ชัด/แยกชิ้น” แบบ funk-pop (kick/snare ชัด มี space)
* Bass-driven composition: ไลน์เบสมี movement เด่น เป็น hook รองของเพลง
* Call-and-response structure: เสียงร้องหลัก ↔ ad-lib/chorus ตอบโต้กัน (สไตล์ performance-driven)
* Minimal but impactful layering: ไม่อัดแน่นเกิน แต่เลือกเสียงที่ “คมและจำได้”
* Percussive vocal integration: เอาเสียงร้อง (breath / hit / ad-lib) มาใช้เป็นส่วนหนึ่งของจังหวะ

Production Style Reference:
* Michael Jackson (groove, rhythm design, vocal-driven production)
* Quincy Jones (arrangement ที่บาลานซ์ groove + musicality)
* Teddy Riley (rhythm + swing + R&B fusion)

Musical Identity:
* Base = modern pop / R&B
* Core engine = groove + rhythm (ได้แรงจาก MJ)
* Texture = soul / funk influence
* Performance link = เพลงถูกออกแบบ “เผื่อเต้น” ตั้งแต่แรก

Dance Skill: Precision Street Hybrid (Animation + Popping based)
* Primary: Animation / Robot / Popping / Waving (สาย illusion—คุมร่างกายละเอียด แยกส่วนชัด)
* Secondary: Lite Breaking + Tricking (ใช้เพิ่ม impact เป็นช่วง ๆ)
* Freestyle: Conceptual / Musical precision (ฟรีสไตล์แบบคิดเป็นภาพ เล่นกับจังหวะหน่วง/เร่ง)
* Signature: Micro-isolation control (แยกส่วนละเอียดมาก), Glitch/Stop-time movement (หยุดเหมือนภาพค้าง/ภาพกระตุก), Delayed hit (ตีจังหวะแบบหน่วงเล็กน้อยให้ดูลึก)
* Added Influence:
    * Emphasis on clean footwork & glide techniques (การใช้เท้าคม ชัด เช่น glide / slide / pivot)
    * Incorporates groove-based stepping with precision timing (สเต็ปเท้ามี groove แต่ยังคุมจังหวะเป๊ะ)
    * Uses footwork as visual contrast to upper-body isolation (ท่อนบน illusion ท่อนล่างเดินจังหวะแบบ smooth)
* Strength: Body control สูงมาก + illusion movement + footwork ที่ “ลื่นแต่คม”
* Limitation: ไม่ใช่สาย power ต่อเนื่องหรือ battle หนัก ๆ (ไม่เน้นแรง/stamina ยาว)
Style Reference:
